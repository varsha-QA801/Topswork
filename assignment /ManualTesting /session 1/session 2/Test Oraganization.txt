1.
List 3 key responsibilities each for a Tester and a Test Leader in a software project, and give one example of how each role contributes during a release cycle.

Answer:
 (A)Tester Responsibilities:
    Write and run test cases.
    Find, report, and retest bugs.
    Make sure the software works as expected.

Example:
A tester opens the shopping app, adds a mobile phone to the cart, and places an order. If the order does not complete, the tester reports the bug. After the developer fixes it, the tester checks it again.

(B)Test Leader Responsibilities:
   Prepare the test plan and testing schedule.
   Assign testing tasks to team members.
   Monitor testing progress and share reports with the project team.

Example:
The Test Leader tells the testing team which features to test first, checks that all important tests are finished, and confirms the software is ready before it is released.
